# 🎲 랜덤 게임 파티

교실에서 바로 쓰는 세 가지 랜덤 결정 게임 (태블릿·전자칠판 최적화)

- ✂️ **가위바위보** — 사람마다 칸을 터치, 터치할 때마다 결과가 바뀜 · 승자 자동 판정
- 🎟️ **쪽지 뽑기** — 10장 중 꽝 하나, 순서대로 뽑다가 꽝이 나오면 폭죽
- 💣 **폭탄 줄 뽑기** — 눈으로 못 따라가게 뒤엉킨 줄, 당기면 불이 타들어감

인원 2~8명, 이름 직접 입력 가능. 별도 설치·인터넷 없이 파일 하나로 동작합니다.

## GitHub Pages로 올리는 법

1. 새 저장소를 만들고 `index.html`(과 `README.md`)을 업로드합니다.
2. 저장소 → **Settings → Pages** 로 이동합니다.
3. **Source** 를 `Deploy from a branch` 로 두고, Branch 를 `main` / 폴더 `/ (root)` 로 선택 후 **Save**.
4. 1~2분 뒤 상단에 뜨는 `https://<아이디>.github.io/<저장소이름>/` 주소로 접속하면 끝!

> 폰트(Pretendard)는 CDN에서 불러오지만, 인터넷이 없어도 시스템 글꼴로 정상 표시됩니다.
